# User Personas

**작성일**: 2026-02-09

## Persona 1: 테이블 고객 (Table Customer)

### 기본 정보
- **역할**: 매장 내 테이블에서 주문하는 고객
- **사용 디바이스**: 테이블에 설치된 태블릿

### 목표
- 빠르고 편리하게 메뉴를 탐색하고 주문
- 주문 상태를 실시간으로 확인
- 직원 호출 없이 자율적으로 주문

### 주요 특성
- 다양한 연령대 (디지털 친숙도 다양)
- 터치 인터페이스 사용
- 짧은 체류 시간 내 빠른 의사결정 필요

### 니즈
- 직관적이고 간단한 UI
- 메뉴 정보 (이미지, 가격, 설명) 명확히 표시
- 주문 실수 방지 (확인 단계)
- 현재 주문 내역 쉽게 확인

### 관련 스토리
- AUTH-C-001 ~ AUTH-C-003 (인증)
- MENU-C-001 ~ MENU-C-003 (메뉴)
- CART-C-001 ~ CART-C-005 (장바구니)
- ORDER-C-001 ~ ORDER-C-004 (주문)
- ERR-C-001 ~ ERR-C-003 (에러)

## Persona 2: 매장 관리자 (Store Admin)

### 기본 정보
- **역할**: 매장 운영 및 주문 관리 담당자
- **사용 디바이스**: PC 또는 태블릿 (관리자 화면)

### 목표
- 실시간으로 들어오는 주문 모니터링
- 주문 상태 관리 및 테이블 운영
- 메뉴 정보 관리

### 주요 특성
- 매장 운영 경험 보유
- 동시에 여러 테이블 관리 필요
- 빠른 의사결정 및 대응 필요

### 니즈
- 한눈에 파악 가능한 대시보드
- 신규 주문 즉시 알림
- 테이블별 주문 현황 파악
- 간편한 메뉴 관리

### 관련 스토리
- AUTH-A-001 ~ AUTH-A-003 (인증)
- ORDMGT-A-001 ~ ORDMGT-A-005 (주문 관리)
- TBLMGT-A-001 ~ TBLMGT-A-005 (테이블 관리)
- MENUMGT-A-001 ~ MENUMGT-A-006 (메뉴 관리)
- ERR-A-001 ~ ERR-A-003 (에러)
